# Crypto Clicker V1
![Gameplay](public/images/gameplay-screenshot.png)
A [clicker game](https://en.wikipedia.org/wiki/Incremental_game) based on cryptocurrency mining built with ReactJS. I like the concept so I'll continue improving this project over time once I manage to sharpen my React skills a little bit more.

The core gameplay consists of you mining/clicking your mining rig in return for cryptocurrency. Which you then use to upgrade your current set up with new components and computers. These upgrades improve your mining speed and while also increasing your electricity costs. Which I later intend to make more problematic as your mining operation grows.
